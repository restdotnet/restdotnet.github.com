Simple .NET HTTP client for RESTful services

Source code 
new RestClient(new Uri("http://server/"))
    .Put<User>("api/users", new UserContract())
    .BadRequest((string error) => Console.WriteLine(error))
    .SuccessAsync(user => Console.WriteLine(user.Id));
Fluent
IRestClient client = new RestClient(new Uri("http://server/"));
IRestRequest request = client.Put<User>("api/users", new UserContract())
request.RegisterCallback<string>(400, error => throw new Exception(error));
User user = await request.ExecuteAsync();
Native

Easy To Use

Simple handling of responses based on http status code. The client will throw typed exception if you didn't handle a code returned by a server.

Cross-framework Compatibility

RestDotNet supports .NET Core, ASP.NET Core, .NET 4.5.1+, Universal Windows apps, and more.

Automatic Deserialization

The correct deserializer is chosen by RestDotNet, based on the content type returned by the server.

Open Source

RestDotNet is open source software and is completely free for commercial use.
